친구들과 정상적으로 실시간 소통을 진행하시려면 구글 Firebase 프로젝트에서 다음의 3가지가 활성화되었는지 꼭 검수해 주세요.

1
Firestore DB 생성 여부

Firebase 콘솔 ➡️ [Firestore Database] 메뉴로 가서 DB 생성을 클릭하셨나요?

2
익명(Anonymous) 인증 켜기

[Authentication] ➡️ [Sign-in method] ➡️ [익명(Anonymous)] 공급자를 사용 설정하셔야 합니다.

3
보안 규칙(Rules) 열어두기

데이터 공유를 위해 Firestore rules 탭을 아래 가이드 규칙으로 교체하셔야 합니다.

💡 보안 규칙 설정 템플릿 복사:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /artifacts/{appId}/public/data/{document=**} { allow read, write: if true; } } }
